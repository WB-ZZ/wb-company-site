주요 기술

  • 컴퓨터 비전 (OpenCV, TensorFlow)
  • 딥러닝 모델 (CNN, YOLO)
  • 실시간 이미지 처리
  • 결함 검출 알고리즘

성과

  • 품질 검사 정확도 97% 달성
  • 검사 시간 70% 단축
  • 연간 30억원 비용 절감
  • 불량률 85% 감소
프로젝트 기간: 2023.03 - 2023.12 (10개월)

신한은행 클라우드 전환

×

프로젝트 개요

신한은행의 레거시 시스템을 AWS 기반 마이크로서비스 아키텍처로 전환하여 확장성과 안정성을 대폭 향상시킨 프로젝트입니다.

주요 기술

  • AWS EKS (Kubernetes)
  • 마이크로서비스 아키텍처
  • API Gateway
  • CI/CD 파이프라인

성과

  • 시스템 응답시간 60% 개선
  • 서버 운영비용 40% 절감
  • 장애 복구시간 90% 단축
  • 동시접속자 5배 증가 대응
프로젝트 기간: 2022.06 - 2023.02 (9개월)

프로젝트 개요

  • Apache Kafka (실시간 스트리밍)

  • Python ML 모델

성과

  • 일 100TB 데이터 실시간 처리
  • 예측 정확도 92% 달성
  • 운영 효율성 55% 개선
  • 의사결정 속도 80% 향상
